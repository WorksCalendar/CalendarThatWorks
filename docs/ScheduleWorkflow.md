# Schedule Workflow

## Entry point

In Timeline view:
- click an employee
- choose:
  - Edit Schedule
  - Request PTO
  - Edit Availability

---

## PTO / Availability

- created from employee actions
- not from generic Add Event
- visible in all calendar views

---

## Automatic logic

If PTO/unavailable overlaps a shift:

1. shift becomes uncovered
2. open shift is created
3. coverage can be assigned

---

## Coverage

- assign another employee
- original shift updates
- covering employee receives mirrored shift
