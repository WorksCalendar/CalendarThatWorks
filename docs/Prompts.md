# Prompt Examples

## Basic prompts

| Use Case | Prompt |
|--------|-------|
| Scheduling | "Create a schedule with employees, PTO, and automatic open shift handling" |
| Dashboard | "Build a calendar with saved views for priority, owner, and status" |
| Timeline | "Create a timeline view with employees and on-call shifts" |
| Coverage | "If an employee is unavailable, create an open shift and allow reassignment" |
| Multi-source | "Combine API events and ICS feeds into one filterable calendar" |

---

## Advanced prompts

| Scenario | Prompt |
|--------|-------|
| Filters | "Add filters for team, tags, and priority with multi-select support" |
| Saved views | "Create saved views for each department" |
| Scheduling templates | "Create recurring on-call schedules for each employee" |
| Custom UI | "Replace the filter bar with a custom component" |

## Goal

Describe your workflow → configure the calendar.
